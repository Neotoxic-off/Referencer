# Referencer
Get all Href and Src from webpage

## Install:
```BASH
git clone https://github.com/Neotoxic-off/Referencer
cd Referencer
pip3 install -r requirements.txt
```

## Usage:

Download ALL:
```BASH
python3 referencer.py
```

Download Specific:
```BASH
python3 referencer.py <extension 1> <extension 2> ...
```

#### Example:

```BASH
python3 referencer.py jpg png wav mp3

# Referencer will download ONLY the jpg, png, wav and mp3 files
```
